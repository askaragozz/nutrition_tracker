# Nutrition Tracker

A Ruby on Rails web application for tracking daily nutrition, meals, and food intake.
Built as a portfolio project to demonstrate mid-level Rails skills with Hotwire.

## Tech Stack
- Ruby 3.4.7
- Rails 7.1.6
- PostgreSQL
- Hotwire (Turbo + Stimulus)
- Bootstrap
- RSpec
- RuboCop
- GitHub Actions (CI)

## Setup

```bash
bundle install
bin/rails db:create db:migrate
bin/rails server
